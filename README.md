# node-caching
How To Setup Caching in Node.js using Redis

**Prerequisites**

To follow through this tutorial you must have the following installed on your computer

* Node JS
* NPM
* Postman/Web Browser
* Code Editor (VsCode)
If you donâ€™t have Node.js installed just head on to the official Node.js website to get a copy of Node.js for your platform. Once you install node.js you will automatically have npm installed.

Instructions:
``javascript
* wget http://download.redis.io/redis-stable.tar.gz
* tar xvzf redis-stable.tar.gz
* cd redis-stable
* make test
* sudo make install
* redis-server
* redis-cli ping
* make
``
